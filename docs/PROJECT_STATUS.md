## 🎯 프로젝트 개요

### 목표
국내 주식 자동매매 시스템 전체 아키텍처 구축
- LS/Kiwoom/타사 API 교체 가능한 Adapter Layer
- 낮은 MDD 기반 전략 개발 및 테스트
- 전략 자동탐색(AutoML) 워크플로우
- Backtesting + Real Trading 통합 HTS
- 웹/모바일 HTS 모니터링 API

### 기술 스택
**백엔드**:
- Python 3.11
- FastAPI
- PostgreSQL
- SQLAlchemy
- LS증권 API

**프론트엔드**:
- React 18.3
- TypeScript 5.6
- Vite 6.0
- Zustand (상태 관리)
- Lightweight Charts
