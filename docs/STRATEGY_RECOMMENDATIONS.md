# 전략 빌더 추천 전략 가이드

## 🤖 AI가 추천하는 전략

### 1. **멀티팩터 모멘텀 전략** (추천도: ⭐⭐⭐⭐⭐)
**개념**: 여러 지표를 조합하여 신뢰도 높은 매수/매도 신호 생성

#### 매수 조건 (모두 만족해야 함)
1. **이동평균 골든크로스**
   - MA(5) > MA(20) (단기선이 장기선 위)
   - 또는 EMA(12) > EMA(26)

2. **모멘텀 확인**
   - RSI < 70 (과매수 아님)
   - RSI > 30 (과매도 아님)
   - MACD cross_above 시그널선

3. **거래량 확인**
   - 거래량 > 거래량 MA(20) (유동성 확보)

4. **추세 강도**
   - ADX > 25 (강한 추세)

#### 매도 조건 (하나라도 만족하면)
1. **이동평균 데드크로스**
   - MA(5) < MA(20)
   - 또는 이동평균선 이탈 (하락) cross_below

2. **과매수 신호**
   - RSI > 70
   - 또는 고점 대비 하락률 > 5%

3. **추세 약화**
   - ADX < 20
   - 또는 연속 음봉 >= 3

**전략 빌더 구현 가능**: ✅ 완전히 가능

---

### 2. **ICT 스마트머니 추종 전략** (추천도: ⭐⭐⭐⭐⭐)
**개념**: 기관투자자의 움직임을 따라가는 전략

#### 매수 조건
1. **BOS 상승 돌파**
   - BOS break_high (lookback: 20)

2. **Order Block 리테스트**
   - Order Block in_block (volume_multiplier: 1.5)

3. **Smart Money Flow 확인**
   - Smart Money Flow bullish

4. **Fair Value Gap 채움**
   - Fair Value Gap in_gap 또는 above_gap

#### 매도 조건
1. **BOS 하락 돌파**
   - BOS break_low (lookback: 20)

2. **Order Block 이탈**
   - Order Block below_block

3. **Smart Money Flow 하락**
   - Smart Money Flow bearish

4. **고점 대비 하락**
   - 고점 대비 하락률 > 3%

**전략 빌더 구현 가능**: ✅ 완전히 가능

---

### 3. **볼린저 밴드 리버설 전략** (추천도: ⭐⭐⭐⭐)
**개념**: 변동성 기반 역추세 전략

#### 매수 조건
1. **하단 밴드 터치**
   - 현재가 < 볼린저 밴드 하단
   - 또는 볼린저 밴드 cross_below 하단

2. **과매도 확인**
   - RSI < 30
   - 또는 스토캐스틱 < 20

3. **거래량 급증**
   - 거래량 > 거래량 MA(20) * 1.5

#### 매도 조건
1. **상단 밴드 터치**
   - 현재가 > 볼린저 밴드 상단
   - 또는 볼린저 밴드 cross_above 상단

2. **과매수 확인**
   - RSI > 70
   - 또는 스토캐스틱 > 80

3. **이익 확보**
   - 고점 대비 하락률 > 2% (트레일링 스탑)

**전략 빌더 구현 가능**: ✅ 완전히 가능

---

### 4. **일목균형표 추세 전략** (추천도: ⭐⭐⭐⭐)
**개념**: 일목균형표의 구름과 전환선/기준선 활용

#### 매수 조건
1. **구름 위 위치**
   - 일목균형표 cloud_above

2. **전환선/기준선 교차**
   - 일목균형표 cross_above (전환선이 기준선 위)

3. **추세 강도**
   - ADX > 25

#### 매도 조건
1. **구름 아래 위치**
   - 일목균형표 cloud_below

2. **전환선/기준선 교차**
   - 일목균형표 cross_below (전환선이 기준선 아래)

3. **연속 음봉**
   - 연속 음봉 >= 3

**전략 빌더 구현 가능**: ✅ 완전히 가능

## 🏢 기관/단체에서 사용하는 실제 시스템매매 전략

### 1. **퀀트 모멘텀 전략** (헤지펀드)
**특징**: 수학적 모델 기반, 다중 지표 조합

#### 핵심 요소
- **팩터 모델**: 여러 기술적 지표를 가중치로 조합
- **리스크 조정**: ATR 기반 포지션 사이징
- **다이버전스 감지**: 가격과 지표의 괴리

#### 전략 빌더 구현
- ✅ 기본 구조는 가능
- ⚠️ 다이버전스 감지는 현재 미지원 (추가 개발 필요)
- ✅ ATR 기반 포지션 사이징은 positionManagement에서 가능

---

### 2. **VWAP 기반 기관 전략** (기관투자자)
**특징**: 거래량 가중 평균가를 기준으로 매매

#### 매수 조건
1. **VWAP 위에서 거래**
   - 현재가 > VWAP

2. **거래량 급증**
   - 거래량 > 거래량 MA(20) * 2.0

3. **모멘텀 확인**
   - RSI 40~60 (중립 구간)

#### 매도 조건
1. **VWAP 아래로 이탈**
   - 현재가 < VWAP

2. **거래량 감소**
   - 거래량 < 거래량 MA(20) * 0.5

**전략 빌더 구현 가능**: ✅ 완전히 가능

---

### 3. **트렌드 팔로잉 전략** (CTA 펀드)
**특징**: 추세가 확실할 때만 진입, 빠른 손절

#### 매수 조건
1. **강한 추세 확인**
   - ADX > 30
   - MA(20) > MA(60) (장기 추세)

2. **돌파 확인**
   - BOS break_high (lookback: 50)

3. **거래량 확인**
   - 거래량 > 거래량 MA(20)

#### 매도 조건
1. **추세 약화**
   - ADX < 20
   - 또는 이동평균선 이탈 (하락)

2. **ATR 기반 손절**
   - ATR * 2.0 이상 하락 시 손절

**전략 빌더 구현 가능**: ✅ 완전히 가능
- ATR 기반 손절은 stopLoss 설정에서 가능

---

### 4. **평균회귀 전략** (시장조성자)
**특징**: 극단적 가격에서 평균으로 회귀 기대

#### 매수 조건
1. **과매도 구간**
   - RSI < 25
   - 스토캐스틱 < 15
   - 볼린저 밴드 하단 터치

2. **지지선 확인**
   - Liquidity Pool near_pool

#### 매도 조건
1. **과매수 구간**
   - RSI > 75
   - 스토캐스틱 > 85
   - 볼린저 밴드 상단 터치

2. **저항선 확인**
   - Liquidity Pool near_pool (고점 클러스터)

**전략 빌더 구현 가능**: ✅ 완전히 가능

---

### 5. **브레이크아웃 전략** (데이 트레이더)
**특징**: 횡보 구간 돌파 시 진입

#### 매수 조건
1. **구조적 돌파**
   - BOS break_high (lookback: 20)

2. **거래량 급증**
   - 거래량 > 거래량 MA(20) * 2.0

3. **볼린저 밴드 확장**
   - ATR > ATR MA(14) * 1.5 (변동성 증가)

#### 매도 조건
1. **이익 실현**
   - 고점 대비 하락률 > 3%

2. **거래량 감소**
   - 거래량 < 거래량 MA(20) * 0.7

**전략 빌더 구현 가능**: ✅ 완전히 가능

## 🎯 전략 빌더 활용 팁

### 1. **단계적 접근**
- 먼저 간단한 전략으로 시작 (예: 골든크로스)
- 백테스트로 성과 확인
- 점진적으로 조건 추가

### 2. **조건 조합**
- 매수 조건은 2~4개가 적당
- 너무 많은 조건은 기회를 놓칠 수 있음
- 매도 조건은 빠르게 반응하는 것이 중요

### 3. **리스크 관리**
- 반드시 손절 설정 (stopLoss)
- 트레일링 스탑 활용
- 포지션 사이징 조절

### 4. **백테스트 필수**
- 최소 1년 이상 데이터로 테스트
- 다양한 시장 환경에서 검증
- 샘플링 편향 주의

   - Order Block in_block

### 고급자용
3. **멀티팩터 모멘텀**

## 📝 전략 등록 방법

1. **전략 빌더 페이지 접속**
2. **전략 정보 입력**
   - 이름: 예) "멀티팩터 모멘텀 전략"
   - 설명: 전략의 핵심 로직 설명

3. **종목 선정 조건 설정**
   - 시가총액, 거래량, PER 등 필터

4. **매수 조건 추가**
   - 위의 추천 조건들을 순서대로 추가

5. **매도 조건 추가**
   - 손실 방지 및 이익 확보 조건

6. **포지션 관리 설정**
   - 손절/익절 비율
   - 트레일링 스탑

7. **저장 및 백테스트**
   - 전략 저장 후 백테스트 실행
   - 성과 확인 후 조정

## ⚠️ 주의사항

1. **과최적화 주의**
   - 과거 데이터에만 맞춘 전략은 실전에서 실패할 수 있음

2. **시장 환경 변화**
   - 전략은 정기적으로 재검토 필요

3. **리스크 관리**
   - 손절은 필수, 감정적 판단 배제

4. **백테스트 한계**
   - 슬리피지, 수수료, 유동성 고려 필요
