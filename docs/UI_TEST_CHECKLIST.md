## ✅ Phase 5: 전략 빌더

### 5.1 페이지 접속
- URL: http://localhost:3000/strategy-builder
- [ ] 페이지 정상 표시
- [ ] 5개 탭 표시
- [ ] 전략 정보 입력 폼 표시

### 5.2 간단한 전략 생성
**전략 정보**:
- 이름: `UI 테스트 전략`
- 설명: `브라우저 UI 테스트용`

**종목 선정**:
- [ ] "종목 선정" 탭 클릭
- [ ] 시가총액 최소: `1000`
- [ ] 거래대금 최소: `1000`
- [ ] 체크박스 작동 확인

**매수 조건**:
- [ ] "매수 조건" 탭 클릭
- [ ] "+ 매수 조건 추가" 버튼 클릭
- [ ] 지표 선택: `이동평균선 (MA)`
- [ ] 기간: `20`
- [ ] 연산자: `>`
- [ ] 값: `MA(50)`

**진입 전략**:
- [ ] "진입 전략" 탭 클릭
- [ ] 진입 방식: `일괄 진입` 선택
- [ ] 라디오 버튼 작동 확인

**매도 조건**:
- [ ] "매도 조건" 탭 클릭
- [ ] "+ 매도 조건 추가" 버튼 클릭
- [ ] 조건 설정

**포지션 관리**:
- [ ] "포지션 관리" 탭 클릭
- [ ] 포지션 사이징: `고정 비율`
- [ ] 포지션 크기: `10%`
- [ ] 최대 포지션 수: `5`
- [ ] 손절 활성화: ✅
- [ ] 손절 비율: `5%`

### 5.3 전략 저장
